                                ~  Open Modes in Python  ~



1) Read Mode:

    * it open the file for read only (No editing no nothing only reading).
    * if the file not exist ( raise : "FileNotFoundError")
    * if you dont have permissiont to read from that file ( raise: "PermissionError")

2) Write Mode:

    * it open a file to Write on it, and it deletes all the existing contents first.
    * if the file was not found it will creat one. (what is the default of permissions that he will have ?)
    * if you dont have permissions (raise : "PermissionError")

3) Append Mode:

    * it only add data to the end of the file it does not delete any thing.
    * if file not found it creat it.
    * if not permission (raise : PermissionError)

4) Creat Mode:

    * it does creat the file, if exit it will fail.
    * if the file exist (raise : FileExistsError)
    * if no permission (raise : PermissionError)

5) And also there is (Read and Write, Write and Read, Append and Read).
